Como importo informação para os temas?
 Previous  Next
A aplicação permite a importação de informação de ficheiros de diferentes tipos.
  • A importação de ficheiros Shapefile, GeoJSON, GeoPackage e KML permite a adição, substituição e/ou a atualização dos elementos dos temas;
  • A importação de ficheiros KMZ e DXF permite a adição e/ou a substituição dos elementos dos temas;
  • A importação de ficheiros CSV permite a atualização da informação alfanumérica dos elementos existentes.

As opções estão disponíveis na figura seguinte.


Importar Shapefiles

A importação de shapefiles é efetuada através da opção "Importar > Shapefile".

É necessário selecionar a shapefile (passo 1 – ficheiros com extensões SHP, DBF, SHX e PRJ), selecionar o sistema de referência da shapefile (passo 2), mapear os campos da shapefile com os da base de dados (passo 3) e, finalmente, selecionar a operação a realizar, de entre as seguintes (passo 4):
  • ADICIONAR os novos elementos da shapefile ao tema;
  • SUBSTITUIR* os elementos existentes no tema pelos da shapefile;
  • ATUALIZAR** os elementos existentes no tema com a informação da shapefile (informação alfanumérica e/ou gráfica).


Importar GeoJSON

A importação de GeoJSONs é efetuada através da opção "Importar > GeoJSON".

É necessário selecionar o GeoJSON (passo 1), selecionar o sistema de referência do ficheiro (passo 2), mapear os campos do GeoJSON com os da base de dados (passo 3) e, finalmente, selecionar a operação a realizar, de entre as seguintes (passo 4):

  • SUBSTITUIR* os elementos existentes no tema pelos do GeoJSON;
  • ATUALIZAR** os elementos existentes no tema com a informação do GeoJSON (informação alfanumérica e/ou gráfica).


Importar GeoPackage

A importação de GeoPackages é efetuada através da opção "Importar > GeoPackage".

É necessário selecionar o GeoPackage (passo 1), selecionar o sistema de referência do ficheiro e a layer a importar (passo 2), mapear os campos do GeoPackage com os da base de dados (passo 3) e, finalmente, selecionar a operação a realizar, de entre as seguintes (passo 4):
  • ADICIONAR os novos elementos do GeoPackage ao tema;
  • SUBSTITUIR* os elementos existentes no tema pelos do GeoPackage;
  • ATUALIZAR** os elementos existentes no tema com a informação da layer do GeoPackage (informação alfanumérica e/ou gráfica).


Importar CSV

Os detalhes dos elementos de um tema podem ser atualizados por importação de ficheiros CSV, através da opção “Importar > CSV”.

É necessário selecionar o CSV (passo 1) e mapear os campos do ficheiro com os da base de dados (passo 2). É necessário indicar qual o campo chave para identificação dos elementos, através da seleção da caixa de verificação existente no lado direito dos campos.

Importar DXF

A importação de DXFs é efetuada através da opção "Importar > DXF".

É necessário selecionar o DXF (passo 1), selecionar o sistema de referência do ficheiro e a(s) layer(s) a importar (passo 2), mapear os campos do DXF com os da base de dados (passo 3) e, finalmente, selecionar a operação a realizar, de entre as seguintes (passo 4):
  • ADICIONAR os novos elementos do GeoPackage ao tema;
  • SUBSTITUIR* os elementos existentes no tema pelos do GeoPackage.

------------------------
* Esta opção está disponível apenas para alguns clientes.
** A atualização dos elementos é efetuada com base no campo ID.
